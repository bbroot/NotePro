# Privacy Policy

**Last updated: June 9, 2026**

---

## Summary

**Note Pro does not collect, store, or transmit any user data.**

---

## Data Storage

- All notes and files created or edited in Note Pro are stored **only on your local device**
- We use your browser's IndexedDB or the local file system (via File System Access API) for persistence
- No data is ever uploaded to external servers

## Personal Information

- Note Pro does **not** require account registration
- We do **not** collect names, email addresses, or any personal information
- We do **not** use cookies or tracking technologies

## Analytics & Third-Party Services

- Note Pro does **not** include any analytics SDKs (Google Analytics, Firebase, etc.)
- Note Pro does **not** include advertising SDKs
- Note Pro does **not** integrate with any third-party services that could collect your data

## Network Access

- Note Pro works **entirely offline**
- No network requests are made to external servers
- The only network activity is for app updates (via App Store or GitHub Releases)

## Your Rights

Since we don't collect any data, there's nothing to:
- Access
- Delete
- Export
- Correct

Your data stays on your device, under your full control.

## Open Source

Note Pro is open source software. You can review the complete source code at:
https://github.com/bbroot/NotePro

## Contact

If you have questions about this policy, please open an issue on GitHub:
https://github.com/bbroot/NotePro/issues

---

**The short version:** Your notes stay on your computer. We never see them, never touch them, never know they exist.
